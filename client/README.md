# Circular project


## Description

A fullstack CRUD with react on front and php/laravel on backend
## Getting Started

### Dependencies

Installing all dependecies first time the project runs (bootstrap, react boostrap, react hook form, react hot toast,axios, uniqId...)

```
cd client
npm install
```

### Executing program

```
cd client
npm start
```
